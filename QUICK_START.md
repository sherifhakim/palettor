# Quick Start Guide - Color Harmony Feature

## What's New?

Your palette generator now creates harmonious color combinations based on color theory! Instead of random colors, you get professionally coordinated palettes using real markers.

## How to Use

### Basic Usage

1. **Open the app** - Navigate to your palette generator
2. **Select a harmony method** - Choose from the new dropdown:
   - **Random** (default) - Picks a different harmony each time
   - **Analogous** - Neighboring colors (calm, cohesive)
   - **Complementary** - Opposite colors (bold, high contrast)
   - **Triadic** - Three evenly spaced colors (balanced, vibrant)
   - **Monochromatic** - Same color family (elegant, subtle)
   - **Split Complementary** - Balanced contrast
   - **Tetradic** - Four-color harmony (rich, diverse)

3. **Click Generate** - Get a harmonious palette!

### Advanced Usage

#### Lock Colors
Want to keep a specific color and build harmony around it?
1. Click the **lock icon** on any color
2. Click **Generate**
3. The palette will be harmonious with your locked color as the base

#### Multiple Locked Colors
- Lock multiple colors to preserve them
- The first locked color determines the harmony base
- Other locked colors stay in place

#### Change Individual Colors
1. Click the **palette icon** on any color
2. Browse and select a specific marker
3. The color will be locked automatically

## Examples

1. Select "Analogous" harmony
2. Generate until you get an orange base
3. Lock the orange
4. Generate again for warm, cohesive colors

### Example 2: Complementary Bold Design
1. Select "Complementary" harmony
2. Generate to get contrasting colors

### Example 3: Monochromatic Elegance
1. Select "Monochromatic" harmony
2. Lock your favorite blue

## Tips

- **Start with Random** (default) to explore different harmony types
- **Lock a color** if you have a specific brand color
- **Try different harmony methods** with the same locked color
- **Analogous** is great for beginners - always looks good!
- **Complementary** is bold - use sparingly
- **Triadic** works well for playful, energetic designs

## Troubleshooting

**Q: Why do I keep getting similar colors?**
A: You might have "Monochromatic" selected. Try "Analogous" or "Triadic" for more variety.

**Q: Can I get more contrast?**
A: Use "Complementary" or "Split Complementary" harmony methods.

**Q: How do I start fresh?**
A: Unlock all colors and click Generate.

**Q: What if I don't like any harmony method?**
A: You can manually select each color using the palette icon on each color card.

## Understanding the Harmony Methods

### Analogous (Neighbors)
```
🔴 Red → 🟠 Orange → 🟡 Yellow
```
Colors next to each other on the color wheel.

### Complementary (Opposites)
```
🔴 Red ↔️ 🟢 Green
🔵 Blue ↔️ 🟠 Orange
```
Colors directly across from each other.

### Triadic (Triangle)
```
🔴 Red - 🟡 Yellow - 🔵 Blue
```
Three colors evenly spaced (120° apart).

### Monochromatic (Same Family)
```
🔵 Light Blue → 🔵 Blue → 🔵 Dark Blue
```
Different shades of the same color.

### Split Complementary (Balanced)
```
🔴 Red + 🟢 Yellow-Green + 🟦 Blue-Green
```
Base color + two colors adjacent to its complement.

### Tetradic (Rectangle)
```
🔴 Red - 🟠 Orange - 🟢 Green - 🔵 Blue
```
Four colors forming a rectangle on the wheel.
