# Color Harmony Examples

## Visual Guide to Harmony Methods

### Color Wheel Reference
```
        0° Red
         │
    330° │ 30°
         │
    300° │ 60° Yellow
         │
    270° │ 90°
         │
    240° │ 120° Green
  Blue   │
    210° │ 150°
         │
    180° Cyan
```

## 1. Analogous Harmony

**Concept**: Colors next to each other on the wheel (±30° intervals)

**Example 1: Red Base (0°)**
```
Base: 0° (Red)
Generated:
  - 0° (Red) ← Base
  - 30° (Red-Orange)
  - 330° (Red-Violet)
  - 60° (Orange)
  - 300° (Violet)

Visual: 🔴 🟠 🟣 🟠 🟣
Effect: Warm, cohesive, harmonious
Use Case: Sunset palettes, warm designs
```

**Example 2: Blue Base (240°)**
```
Base: 240° (Blue)
Generated:
  - 240° (Blue) ← Base
  - 270° (Blue-Violet)
  - 210° (Blue-Green)
  - 300° (Violet)
  - 180° (Cyan)

Visual: 🔵 🟣 🟦 🟣 🟦
Effect: Cool, calming, unified
Use Case: Ocean themes, professional designs
```

## 2. Complementary Harmony

**Concept**: Opposite colors on the wheel (180° apart)

**Example 1: Red Base (0°)**
```
Base: 0° (Red)
Generated:
  - 0° (Red) ← Base
  - 180° (Cyan)
  - 180° (Cyan)
  - 180° (Cyan)
  - 180° (Cyan)

Visual: 🔴 🟦 🟦 🟦 🟦
Effect: High contrast, vibrant, bold
Use Case: Logos, attention-grabbing designs
```

**Example 2: Yellow Base (60°)**
```
Base: 60° (Yellow)
Generated:
  - 60° (Yellow) ← Base
  - 240° (Blue)
  - 240° (Blue)
  - 240° (Blue)
  - 240° (Blue)

Visual: 🟡 🔵 🔵 🔵 🔵
Effect: Classic contrast, energetic
Use Case: Sports teams, bold branding
```

## 3. Triadic Harmony

**Concept**: Three colors evenly spaced (120° apart)

**Example 1: Red Base (0°)**
```
Base: 0° (Red)
Generated:
  - 0° (Red) ← Base
  - 120° (Green)
  - 240° (Blue)
  - 360° → 0° (Red)
  - 480° → 120° (Green)

Visual: 🔴 🟢 🔵 🔴 🟢
Effect: Balanced, vibrant, playful
Use Case: Children's designs, festive themes
```

**Example 2: Orange Base (30°)**
```
Base: 30° (Orange)
Generated:
  - 30° (Orange) ← Base
```


## 5. Split Complementary Harmony

**Concept**: Base + two colors adjacent to complement (150° and 210°)

```
Effect: Balanced contrast, less tension than complementary
Use Case: Versatile designs, web interfaces
```

**Example 2: Blue Base (240°)**

## Tips for Best Results

### Analogous
- Best with 3-5 colors
- Works great for nature themes
- Try locking a warm or cool base

### Complementary
- Use sparingly (high contrast)
- Great for 2-3 colors
- Lock your brand color as base

### Triadic
- Perfect for 3 colors
- Adds energy and playfulness
- Works well with primary colors

### Monochromatic
- Excellent for 4-7 colors
- Professional and elegant
- Lock your favorite color

### Split Complementary
- Versatile for any size
- Less intense than complementary
- Good for web design

### Tetradic
- Best with 4-5 colors
- Rich and complex
- Great for artistic projects

## Common Patterns

### Warm Palette (Analogous)
```
Lock: Orange (30°)
Result: Red-Orange-Yellow range
Use: Autumn, warmth, energy
```

### Cool Palette (Analogous)
```
Lock: Blue (240°)
Result: Cyan-Blue-Violet range
Use: Winter, calm, professional
```

### High Contrast (Complementary)
```
Lock: Any color
Result: Color + its opposite
Use: Logos, bold statements
```

### Balanced (Triadic)
```
Lock: Any primary color
Result: Evenly distributed colors
Use: Playful, balanced designs
```

### Sophisticated (Monochromatic)
```
Lock: Any color
Result: Shades of same color
Use: Elegant, minimalist designs
```
